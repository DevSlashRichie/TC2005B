# TC2005B - Construcción de Software y Toma de Decisiones

Este repositorio contiene las actividades y laboratorios para la materia TC2005B.

## Tecnologías Utilizadas
- **HTML5:** Estructura y semántica.
- **CSS3:** Estilos personalizados y diseño responsivo.
- **JavaScript (ES6+):** Lógica de programación y manipulación del DOM.
- **Tailwind CSS:** Framework de CSS para el diseño rápido de interfaces (utilizado en Lab 2).
- **Neovim:** Editor de texto principal.
